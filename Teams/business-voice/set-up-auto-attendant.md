# <a name="step-6-set-up-an-auto-attendant-for-your-companys-main-phone-number"></a>Schritt 6: Einrichten einer automatischen Telefonwarte für die Haupttelefonnummer Ihres Unternehmens

Mit einer automatischen Telefonkonferenz können Sie ein Anrufmenü einrichten, damit Anrufer Ihrer Haupttelefonnummer aus einer Liste der Optionen auswählen können. Sie könnten beispielsweise Optionen haben, um eine Verbindung mit einer Vertriebs- oder Supportabteilung, einer bestimmten Person, der Schalttafel und sogar einer anderen automatischen Attendant herzustellen.
