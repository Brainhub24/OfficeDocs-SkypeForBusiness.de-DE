# <a name="phone-number-management-for-latvia"></a>Telefon der Nummerverwaltung für Lettland

In dieser Tabelle finden Sie Informationen zum Abrufen und Verwalten von Telefonnummern in Lettland für Microsoft Teams.
  
Weitere Informationen finden Sie unter [Verwalten von Telefonnummern für Ihre Organisation](manage-phone-numbers-for-your-organization.md).
  
|**Was möchten Sie tun?**|**Laden Sie die richtigen Formulare herunter.**|
|:-----|:-----|
|Rufen Sie neue Benutzer- oder Diensttelefonnummern ab, die nicht im Microsoft Teams Admin Center aufgeführt sind. |In Lettland können Sie Telefonnummern für Ihre Organisation über das Admin Center Microsoft Teams erhalten. Sie können auch eine Anforderung für eine neue Telefonnummer übermitteln, indem Sie das Formular herunterladen und sich an [den TNS Service Desk wenden, um Anweisungen zu erhalten.](contact-tns-service-desk.md)<br/><br/>[Anforderung einer neuen Telefonnummer für Lettland (v.1.0).](<https://download.microsoft.com/download/7/4/7/7477191d-1b6d-4d6b-a97f-c958ae204db5/New phone number request for Latvia (v.1.0) (en-us).pdf>) <br/><br/> Bevor Ihnen Telefonnummern zugewiesen werden können, müssen Sie das Formular Wissen Sie Ihren Kunden [herunter,](<https://download.microsoft.com/download/1/8/d/18d3c082-984f-4271-ba23-80225904c5e2/Latvia - Know Your Customer.pdf>)es vollständig ausfüllen und an uns senden.<br/><br/>Es kann 1 bis 4 Wochen dauern, bis neue Bestellungen in diesem Markt eins bis vier Wochen dauern, je nachdem, ob es in der jeweiligen Region Aktien gibt. |
|Lassen Sie Telefonnummern an uns übertragen. | Laden Sie dieses Formular herunter. <br/><br/>[Genehmigungsschreiben für Lettland (alle Nummern) (v.1.0)](<https://download.microsoft.com/download/d/d/6/dd67eefd-1886-44fd-a845-127e5cadc1ee/Latvia LOA.pdf>) <br/><br/>Füllen Sie das Feld aus, und wenden Sie sich [an den TNS Service Desk, um Anweisungen zu erhalten.](contact-tns-service-desk.md) Sie müssen auch eine Rechnung für das Hilfsprogramm von der Geschäftsadresse angeben, für die Sie Nummern übertragen möchten.<br/><br/>Die Übertragung von Nummern kann in diesem Markt bis zu 3 Wochen dauern. Diese Nummern können derzeit nicht für Konferenzen verwendet werden.|
|Sie möchten Telefonnummern im nationalen Telefonverzeichnis veröffentlichen bzw. nicht mehr veröffentlichen.  <br/> |[Wenden Sie sich an den TNS Service Desk,](contact-tns-service-desk.md) um Anweisungen zu der Liste der Telefonnummern zu erhalten, die Sie veröffentlichen bzw. deren Veröffentlichung aufheben möchten. <br/> |
|Sie möchten die Telefonnummer eines Benutzers in eine Dienstnummer ändern.  <br/> |[Wenden Sie sich an den TNS Service Desk,](contact-tns-service-desk.md) um Anweisungen zu der Liste der Benutzernummern zu erhalten, die Sie in Servicenummern umwandeln möchten. <br/> |
|Durchführen von Verzeichnisdienstupdates für Nummern von Anrufwarteschleifen oder automatischen Telefonzentralen|Anweisungen zum Hinzufügen oder Ändern von Rufnummern in der Anrufwarteschleife oder der automatischen Telefonwarteschleife erhalten Sie [beim TNS Service Desk.](contact-tns-service-desk.md) |
