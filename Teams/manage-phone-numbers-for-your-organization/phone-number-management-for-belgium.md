# <a name="phone-number-management-for-belgium"></a>Telefonnummernverwaltung für Belgien

In dieser Tabelle finden Sie Informationen zum Erhalten und Verwalten von Telefonnummern in Belgien für Microsoft Teams.
  
Weitere Informationen finden Sie unter [Verwalten von Telefonnummern für Ihre Organisation](manage-phone-numbers-for-your-organization.md).
  
|**Was möchten Sie tun?**|**Laden Sie die richtigen Formulare herunter, und senden Sie sie an uns.**|
|:-----|:-----|
|Neue Telefonnummern für Benutzer und Dienste erhalten, die im Admin Center Microsoft Teams werden.   <br/> |In Belgien können Sie keine Telefonnummern für Ihre Organisation über das Microsoft Teams Admin Center erhalten, sondern müssen eine Anforderung für eine neue Telefonnummer übermitteln. <br/><br/>Laden Sie eines dieser Formulare herunter: <br/>  [Anforderung einer neuen Telefonnummer für Belgien (v.5.0) (de-be)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/new-number-request-forms/new-phone-number-request-for-belgium-(v.5.0)-(de-be).pdf?raw=true) <br/> [Anforderung einer neuen Telefonnummer für Belgien (v.5.0) (fr-be)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/new-number-request-forms/new-phone-number-request-for-belgium-(v.5.0)-(fr-be).pdf?raw=true) <br/> [Anforderung einer neuen Telefonnummer für Belgien (v.5.0) (nl-be)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/new-number-request-forms/new-phone-number-request-for-belgium-(v.5.0)-(nl-be).pdf?raw=true) <br/>   [Anforderung einer neuen Telefonnummer für Belgien (v.5.0) (en-us)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/new-number-request-forms/new-phone-number-request-for-belgium-(v.5.0)-(en-us).pdf?raw=true)<br/><br/>Füllen Sie das Feld aus, und wenden Sie sich [an den PTSN Service Desk, um Anweisungen zu erhalten.](contact-pstn-service-desk.md) <br/>|
|Lassen Sie Telefonnummern an uns übertragen.  <br/> | Laden Sie eines dieser Formulare herunter. <br/>[Genehmigungsschreiben für Belgien (alle Nummern) (v.5.0) (nicht be)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/LOA-forms/letter-of-authorization-(loa)-for-belgium-(all-numbers)-(v.5.0)-(de-be).pdf) <br/> [Genehmigungsschreiben für Belgien (alle Nummern) (v.6.1) (fr-be)](https://download.microsoft.com/download/d/f/3/df304a78-db38-41f9-98d2-0b590cf8c68f/letter-of-authorization-(loa)-for-belgium-(all-numbers)-(v.6.1)-(fr-be).pdf)<br/> [Genehmigungsschreiben für Belgien (alle Nummern) (v.6.2) (nl-be)](https://download.microsoft.com/download/4/b/a/4bacc856-3bf2-45f7-ade2-8b1d81fc09c6/letter-of-authorization-(loa)-for-belgium-(all-numbers)-(v6.2)-(nl-be).pdf) <br/>   <br/>   Füllen Sie das Formular aus, und [wenden Sie sich bezüglich Anweisungen an PTSN](contact-pstn-service-desk.md). <br/> |
|Sie möchten Telefonnummern im nationalen Telefonverzeichnis veröffentlichen bzw. nicht mehr veröffentlichen.  <br/> |[Wenden Sie sich an den PTSN Service Desk,](contact-pstn-service-desk.md) um Anweisungen zu der Liste der Telefonnummern zu erhalten, die Sie veröffentlichen bzw. deren Veröffentlichung aufheben möchten.      <br/> |
|Sie möchten die Telefonnummer eines Benutzers in eine Dienstnummer ändern.  <br/> |[Wenden Sie sich an den PTSN Service Desk,](contact-pstn-service-desk.md) um Anweisungen zu der Liste der Benutzernummern zu erhalten, die Sie in Servicenummern umwandeln möchten. <br/> |
|Durchführen von Verzeichnisdienstupdates für Nummern von Anrufwarteschleifen oder automatischen Telefonzentralen|Anweisungen zum Hinzufügen oder Ändern von Rufnummern in der Anrufwarteschleife oder der automatischen Telefonkonferenz erhalten Sie [beim PSTN Service Desk.](contact-pstn-service-desk.md) |
