# <a name="phone-number-management-for-belgium"></a>Telefonnummernverwaltung für Belgien

In dieser Tabelle finden Sie Informationen zum Erhalten und Verwalten von Telefonnummern in Belgien für Microsoft Teams.
  
Weitere Informationen finden Sie unter [Verwalten von Telefonnummern für Ihre Organisation](manage-phone-numbers-for-your-organization.md).
  
|**Was möchten Sie tun?**|**Laden Sie die richtigen Formulare herunter, und senden Sie sie an uns.**|
|:-----|:-----|
|Neue Telefonnummern für Benutzer und Dienste erhalten, die nicht im Admin Center Microsoft Teams werden.   <br/> |In Belgien können Sie keine Telefonnummern für Ihre Organisation über das Microsoft Teams Admin Center erhalten, sondern müssen eine Anforderung für eine neue Telefonnummer übermitteln. <br/><br/>Laden Sie eines dieser Formulare herunter: <br/>  [Anforderung einer neuen Telefonnummer für Belgien (v.5.0) (de-be)](https://download.microsoft.com/download/7/a/2/7a200ded-8377-48af-be13-672e32f9ed6b/new-phone-number-request-for-belgium-(v.5.0)-(de-be).pdf) <br/> [Anforderung einer neuen Telefonnummer für Belgien (v.5.0) (fr-be)](https://download.microsoft.com/download/6/8/9/689c1af5-b7cc-4f38-bade-02343172efcc/new-phone-number-request-for-belgium-(v.5.0)-(fr-be).pdf) <br/> [Anforderung einer neuen Telefonnummer für Belgien (v.5.0) (nl-be)](https://download.microsoft.com/download/3/0/0/300dbedd-a6cc-49c1-a6c0-a8653ac8e869/new-phone-number-request-for-belgium-(v.5.0)-(nl-be).pdf) <br/>   [Anforderung einer neuen Telefonnummer für Belgien (v.5.0) (en-us)](https://download.microsoft.com/download/9/e/4/9e4f9a33-5a29-4409-a601-f489c84a3c32/new-phone-number-request-for-belgium-(v.5.0)-(en-us).pdf)<br/><br/>Füllen Sie das Feld aus, und wenden Sie sich [an den TNS Service Desk, um Anweisungen zu erhalten.](contact-tns-service-desk.md) <br/>|
|Lassen Sie Telefonnummern an uns übertragen.  <br/> | Laden Sie eines dieser Formulare herunter. <br/>[Genehmigungsschreiben für Belgien (alle Nummern) (v.5.0) (nicht be)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/LOA-forms/letter-of-authorization-(loa)-for-belgium-(all-numbers)-(v.5.0)-(de-be).pdf) <br/> [Genehmigungsschreiben für Belgien (alle Nummern) (v.6.1) (fr-be)](https://download.microsoft.com/download/d/f/3/df304a78-db38-41f9-98d2-0b590cf8c68f/letter-of-authorization-(loa)-for-belgium-(all-numbers)-(v.6.1)-(fr-be).pdf)<br/> [Genehmigungsschreiben für Belgien (alle Nummern) (v.6.2) (nl-be)](https://download.microsoft.com/download/4/b/a/4bacc856-3bf2-45f7-ade2-8b1d81fc09c6/letter-of-authorization-(loa)-for-belgium-(all-numbers)-(v6.2)-(nl-be).pdf) <br/>   <br/>   Füllen Sie das Feld aus, und wenden Sie sich an [den TNS Service Desk, um Anweisungen zu erhalten.](contact-tns-service-desk.md) <br/> |
|Sie möchten Telefonnummern im nationalen Telefonverzeichnis veröffentlichen bzw. nicht mehr veröffentlichen.  <br/> |[Wenden Sie sich an den TNS Service Desk,](contact-tns-service-desk.md) um Anweisungen zu der Liste der Telefonnummern zu erhalten, die Sie veröffentlichen bzw. deren Veröffentlichung aufheben möchten.      <br/> |
|Sie möchten die Telefonnummer eines Benutzers in eine Dienstnummer ändern.  <br/> |[Wenden Sie sich an den TNS Service Desk,](contact-tns-service-desk.md) um Anweisungen zu der Liste der Benutzernummern zu erhalten, die Sie in Servicenummern umwandeln möchten. <br/> |
|Durchführen von Verzeichnisdienstupdates für Nummern von Anrufwarteschleifen oder automatischen Telefonzentralen|Anweisungen zum Hinzufügen oder Ändern von Rufnummern in der Anrufwarteschleife oder der automatischen Telefonwarteschleife erhalten Sie [beim TNS Service Desk.](contact-tns-service-desk.md) |
