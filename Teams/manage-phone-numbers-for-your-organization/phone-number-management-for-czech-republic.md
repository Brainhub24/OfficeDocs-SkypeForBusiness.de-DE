# <a name="phone-number-management-for-the-czech-republic"></a>Telefonnummernverwaltung für die Tschechische Republik

[!INCLUDE [preview-feature](../includes/preview-feature.md)]

In dieser Tabelle finden Sie Informationen zum Abrufen und Verwalten von Telefonnummern in der Tschechischen Republik für Microsoft Teams.

Weitere Informationen finden Sie unter [Verwalten von Telefonnummern für Ihre Organisation](manage-phone-numbers-for-your-organization.md).

|**Was möchten Sie tun?**|**Laden Sie die richtigen Formulare herunter.**|
|:-----|:-----|
|Rufen Sie neue Benutzer- oder Diensttelefonnummern ab, die nicht im Microsoft Teams Admin Center aufgeführt sind. |In der Tschechischen Republik können Sie Telefonnummern für Ihre Organisation über das Microsoft Teams Admin Center abrufen. Sie können auch eine neue Telefonnummernanfrage übermitteln, indem Sie das Formular "Ihr Kunde kennen" herunterladen und [sich an den TNS Service Desk wenden, um Entsprechendes zu erfahren](contact-tns-service-desk.md).<br/><br/>Bevor Ihnen die Nummern zugewiesen werden können, müssen Sie ihr Kundenformular herunterladen: <br/><br/><ul><li>[Kennen Sie Ihr Kundenformular für Die Tschechische Republik](https://download.microsoft.com/download/6/f/6/6f66c134-5ad4-42d8-b7f9-f857fb5fd4ca/KYC-CZ-CZ.docx)</ul></li> <br/>Neue Bestellungen können zwei bis drei Wochen dauern, um diesen Markt zu füllen.
|Lassen Sie Telefonnummern an uns übertragen. | Laden Sie dieses Formular herunter. <br/><br/>[Schriftliche Vollmacht für die Tschechische Republik (alle Nummern) (v.2.0) (cz-cz)](https://download.microsoft.com/download/6/f/6/6f66c134-5ad4-42d8-b7f9-f857fb5fd4ca/LOA-CZ-GEO-EN.pdf)<br/><br/>Füllen Sie sie aus, und [wenden Sie sich an den TNS Service Desk, um Entsprechendes zu erhalten](contact-tns-service-desk.md). Sie müssen auch die aktuellste Rechnung ihres derzeitigen Betreibers angeben, damit die Nummer(en) überwiesen werden können.<br/><br/>Die Übertragung von Nummern kann in diesem Markt zwei bis drei Wochen dauern. <br/><br/>Diese Nummern können zurzeit nicht für Konferenzen verwendet werden. |
|Sie möchten Telefonnummern im nationalen Telefonverzeichnis veröffentlichen bzw. nicht mehr veröffentlichen.  <br/> |[Wenden Sie sich an den TNS Service Desk, um Anweisungen](contact-tns-service-desk.md) zur Liste der Telefonnummern zu erhalten, die Sie veröffentlichen/die Veröffentlichung aufheben möchten. <br/> |
|Sie möchten die Telefonnummer eines Benutzers in eine Dienstnummer ändern.  <br/> |[Wenden Sie sich an den TNS Service Desk, um Anweisungen](contact-tns-service-desk.md) zur Liste der Benutzernummern zu erhalten, die Sie in Servicenummern konvertieren möchten. <br/> |
|Durchführen von Verzeichnisdienstupdates für Nummern von Anrufwarteschleifen oder automatischen Telefonzentralen|Wenn Sie Anrufwarteschleifen- oder automatische Telefonzentralennummern hinzufügen oder ändern möchten, wenden Sie sich [an den TNS Service Desk, um Anweisungen zu erhalten](contact-tns-service-desk.md). |
