# <a name="phone-number-management-for-italy"></a>Telefon der Nummerverwaltung für Italien

In dieser Tabelle finden Sie Informationen zum Erhalten und Verwalten von Telefonnummern in Italien für Microsoft Teams.
  
Weitere Informationen finden Sie unter [Verwalten von Telefonnummern für Ihre Organisation](manage-phone-numbers-for-your-organization.md).
  
|**Was möchten Sie tun?**|**Laden Sie die richtigen Formulare herunter, und senden Sie sie an uns.**|
|:-----|:-----|
|Rufen Sie neue Benutzer- oder Diensttelefonnummern ab, die nicht im Microsoft Teams Admin Center aufgeführt sind.   <br/> |Laden Sie eines dieser Formulare herunter. <br>  [Anforderung einer neuen Telefonnummer für Italien (v.5.0) (it-it)](https://download.microsoft.com/download/7/2/7/727217df-80e4-4ff5-b9ed-65b84cdceba6/new-phone-number-request-for-italy-(v.5.0)-(it-it).pdf) <br/>  [Anforderung einer neuen Telefonnummer für Italien (v.5.0) (en-us)](https://download.microsoft.com/download/4/1/7/417d2578-3686-4530-87ea-e9f36e8653c1/new-phone-number-request-for-italy-(v.5.0)-(en-us).pdf)<br/>  <br/>Füllen Sie das Feld aus, [und wenden Sie sich an TNS, um Anweisungen zu erhalten.](contact-tns-service-desk.md) |
|Lassen Sie Telefonnummern an uns übertragen.  <br/> | Laden Sie dieses Formular herunter. <br/>[Genehmigungsschreiben für Italien (alle Nummern) (v.6.0) (it-it)](https://download.microsoft.com/download/e/8/a/e8a20665-cbea-4f15-a50c-b8e1f6ef352d/letter-of-authorization-(loa)-for-italy-(all-numbers)-(v.6.0)-(it-it).pdf) <br/> <br/>Füllen Sie das Feld aus, und wenden Sie sich [an den TNS Service Desk, um Anweisungen zu erhalten.](contact-tns-service-desk.md) |
|Sie möchten Telefonnummern im nationalen Telefonverzeichnis veröffentlichen bzw. nicht mehr veröffentlichen.  <br/> |[Wenden Sie sich an den TNS Service Desk,](contact-tns-service-desk.md) um Anweisungen zu der Liste der Telefonnummern zu erhalten, die Sie veröffentlichen bzw. deren Veröffentlichung aufheben möchten.      <br/> |
|Sie möchten die Telefonnummer eines Benutzers in eine Dienstnummer ändern.  <br/> |[Wenden Sie sich an den TNS Service Desk,](contact-tns-service-desk.md) um Anweisungen zu der Liste der Benutzernummern zu erhalten, die Sie in Servicenummern umwandeln möchten. <br/> |
|Durchführen von Verzeichnisdienstupdates für Nummern von Anrufwarteschleifen oder automatischen Telefonzentralen|Laden Sie zum Hinzufügen oder Ändern von Rufnummern in der Anrufwarteschleife oder automatischen Telefon attendant dieses Formular für Verzeichnisdienste für Italien [(v.1.0) (it-it)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/directory-services-form-for-italy-(v.1.0)-(it-it).pdf?raw=true)herunter, füllen Sie es aus, und wenden Sie sich an [TNS,](contact-tns-service-desk.md)um Anweisungen zu erhalten. |
|Überprüfen des Dienstleistungscharts| Laden Sie den [Dienstleistungscharts für Italien (v.1.0) (it-it)](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/charter-of-services-for-italy-(v.1.0)-(it-it).pdf?raw=true)herunter, und lesen Sie ihn. |
