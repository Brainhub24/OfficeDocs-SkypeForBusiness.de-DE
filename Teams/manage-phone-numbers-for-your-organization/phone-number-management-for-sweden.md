# <a name="phone-number-management-for-sweden"></a>Telefon der Nummerverwaltung für Schweden

In dieser Tabelle finden Sie Informationen zum Erhalten und Verwalten von Telefonnummern in Schweden für Microsoft Teams.
  
Weitere Informationen finden Sie unter [Verwalten von Telefonnummern für Ihre Organisation](manage-phone-numbers-for-your-organization.md).

Übertragen Sie uns Ihre Telefonnummern.  <br/> |Laden Sie dieses Formular herunter. <br/>[Genehmigungsschreiben für Schweden (alle Nummern) (v.5.0) (sv-se)](https://download.microsoft.com/download/b/d/0/bd030c0c-67c5-40fb-a148-d12a4675aa79/letter-of-authorization-(loa)-for-sweden-(all-numbers)-(v.5.0)-(sv-se).pdf)<br/><br/> Füllen Sie das Formular aus, und [senden Sie es an uns](mailto:ptneu@microsoft.com). <br/> |
