# <a name="microsoft-teams-add-on-licenses"></a>Add-On-Lizenzen für Microsoft Teams

## <a name="what-features-can-i-get-with-add-on-licenses"></a>Welche Features kann ich über Add-On-Lizenzen erhalten?

Add-On-Lizenzen sind Lizenzen für bestimmte Funktionen von Microsoft Teams. Sie bieten Ihnen die Flexibilität, Features nur für Benutzer in Ihrer Organisation hinzuzufügen, die sie benötigen. Um eine Funktion hinzuzufügen, kaufen Sie eine Add-On-Lizenz für jeden Benutzer, der diese Funktion verwenden wird.

Welche Add-On-Lizenzierungsoptionen für Sie verfügbar sind, hängt von Ihrem Microsoft 365- oder Office 365-Plan ab. Wenn Sie wissen möchten, welche Sprachoptionen für Sie verfügbar sind, lesen Sie den Abschnitt Welche Sprachfeatures sind in [meinem](#what-voice-features-are-available-with-my-plan) Plan verfügbar?.

Hier finden Sie eine Liste Teams Add-On-Lizenzen und deren Features, die Sie Ihrem Abonnement Teams können.

## <a name="what-voice-features-are-available-with-my-plan"></a>Welche VoIP-Funktionen stehen in meinem Plan zur Verfügung?

Sehen Sie sich an, welche Add-On-Lizenzen Sie benötigen, um VoIP-Funktionen in Microsoft Teams je nach bestehendem Plan zu erhalten.

Je nachdem, welche VoIP-Funktionen Sie wünschen, empfehlen wir, zu vergleichen, ob es für Sie kostengünstiger ist, zu einem Plan zu wechseln, in dem diese Funktionen enthalten sind. Beispielsweise führt der Kauf eines Softwarepakets im Microsoft 365 Enterprise-Plan für die meisten großen Organisationen zu geringeren Gesamtkosten. Üblicherweise sind die Gesamtkosten höher, wenn Sie mehrere Add-On-Lizenzen einzeln statt als Teil eines Plans kaufen.

#### <a name="small-and-medium-businesses"></a>[**Kleine und mittelständische Unternehmen**](#tab/small-business/)

Wenn Sie für ein kleines oder mittelgroßes Unternehmen (bis zu 300 Benutzer) tätig sind, stehen Ihnen vier Möglichkeiten zur Verfügung, um VoIP-Funktionen zu erhalten:

- Kaufen Sie einen Microsoft 365 Business-Plan und Microsoft 365 Teams Telefon mit dem Anrufplanpaket.
- Kaufen Sie einen Enterprise E1- oder E3-Plan Microsoft 365 Teams Telefon mit dem Anrufplanpaket.
- Kauf eines Enterprise E1- oder E3-Plans und Hinzufügen einzelner VoIP-Funktionen.
- Kauf eines Enterprise E5-Plans, der VoIP-Funktionen umfasst.
  - Der Microsoft 365 Enterprise E5-Plan enthält Teams Telefon, Organisationen müssen jedoch weiterhin eine [PSTN-Konnektivitätsoption auswählen](../pstn-connectivity.md).

Für Microsoft 365 Teams Telefon haben Sie zwei Optionen zur Auswahl:

- **Microsoft 365 Teams Telefon mit Anrufplanpaket**: Rufen Sie diese Informationen ab, wenn Sie möchten, dass die Verwaltung und Abrechnung von Telefonnummern von Microsoft abgewickelt wird. Überprüfen Sie, [ob Teams Telefon Anrufplans in Ihrer Region verfügbar ist](../country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md).
- **Microsoft 365 Teams Telefon eigenständige Lizenz**: Diese Lizenz enthält Teams Telefon, aber Sie müssen weiterhin eine Verbindung mit dem PSTN herstellen. Weitere Informationen zu Sprachlösungen finden Sie unter [Planen Teams Sprachlösung](../cloud-voice-landing-page.md). Informationen zum Anzeigen aller PSTN-Konnektivitätsoptionen finden Sie unter [PstN-Konnektivitätsoptionen](../pstn-connectivity.md).

Weitere Informationen finden Sie Teams Telefon [Anleitungen für kleine und mittelständische Unternehmen](../business-voice/whats-business-voice.md).
