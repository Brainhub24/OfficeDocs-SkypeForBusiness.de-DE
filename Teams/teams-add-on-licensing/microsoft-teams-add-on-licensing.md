# <a name="microsoft-teams-add-on-licenses"></a>Add-On-Lizenzen für Microsoft Teams

## <a name="which-capabilities-can-i-get-with-add-on-licenses"></a>Welche Funktionen kann ich mit Add-On-Lizenzen erhalten?

Add-On-Lizenzen sind Lizenzen für bestimmte Microsoft Teams Funktionen. Sie bieten Ihnen die Flexibilität, Funktionen nur für Benutzer in Ihrer Organisation hinzuzufügen, die diese benötigen. Um eine Funktion hinzuzufügen, kaufen Sie eine Add-On-Lizenz für jeden Benutzer, der sie verwenden wird.

Die verfügbaren Add-On-Lizenzierungsoptionen hängen von Ihrem Microsoft 365 Plan ab. Wenn Sie wissen möchten, welche Optionen für Sprachanrufe und Audiokonferenzen für Sie verfügbar sind, lesen Sie den Abschnitt Welche [Microsoft-Funktionen](#which-microsoft-capabilities-are-available-with-my-plan) sind mit meinem Plan verfügbar? in diesem Artikel.

## <a name="which-microsoft-capabilities-are-available-with-my-plan"></a>Welche Microsoft-Funktionen sind in meinem Plan verfügbar?

Alle Microsoft 365 ermöglichen es Teams Benutzern, innerhalb Ihrer Organisation mit VoIP-Technologie Peer-to-Peer-Anrufe zu telefonieren. Möglicherweise möchten Sie jedoch verschiedene Lizenzen für Funktionen wie Anrufe bei PSTN-Nummern oder Einrichten einer Telefonanlage hinzufügen.

### <a name="audio-conferencing"></a>Audiokonferenz

Sie können **Lizenzen für Audiokonferenzen** kostenlos zu allen Abonnements Microsoft 365 hinzufügen.

Microsoft 365 E5/A5/G5-Pläne umfassen Lizenzen für Audiokonferenzen, daher müssen Sie bei diesen Plänen keine zusätzlichen Lizenzen für Audiokonferenzen erwerben.

### <a name="voice-and-pstn-connectivity"></a>Sprach- und PSTN-Anbindung

Ganz gleich, ob Sie ein kleines Unternehmen, eine Unternehmensorganisation, eine Bildungseinrichtung, eine Behörde oder eine gemeinnützige Organisation sind, Sie haben drei Möglichkeiten, um Teams Telefonsystem und PSTN-Anbindung zu erhalten:

- Kaufen Sie Microsoft 365 Business-, E1/E3-, A1/A3- oder G1/G3-Plan,  und fügen Sie Teams Telefon mit Anrufplan-Bündellizenzen hinzu.
  - Dieses Modell bietet Ihnen Teams Telefonsystem PBX-Funktionen und einen Microsoft-Anrufplan für Inland anrufe.
  - Wählen Sie dieses Setup aus, wenn die Verwaltung und Abrechnung aller Telefonnummern durch Microsoft abgewickelt werden soll.
  - Überprüfen Sie, ob [Teams Telefon mit einer Anrufplan-Bündellizenz in Ihrer Region verfügbar ist](../country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md).

- Kaufen Sie einen E1/E3-, A1/A3- oder G1/G3-Plan, fügen Sie **Teams Telefon Standard-Lizenzen** hinzu, und wählen Sie einen eigenständigen Microsoft-Anrufplan oder einen PSTN-Netzbetreiber von Drittanbietern aus.
  - Dieses Modell gibt Ihnen Teams Telefonsystem, aber Sie müssen weiterhin eine Verbindung mit dem PSTN herstellen.
  - Informationen zum Anzeigen aller PSTN-Konnektivitätsoptionen finden Sie unter [PstN-Konnektivitätsoptionen](../pstn-connectivity.md).
  - Weitere Informationen zu allen Sprachlösungen finden Sie unter [Planen Teams Sprachlösung](../cloud-voice-landing-page.md).

- Kaufen Sie einen E5/A5/G5-Plan, und fügen Sie Microsoft-Anrufpläne hinzu, oder stellen Sie eine Verbindung mit einem PstN-Konnektivitätsanbieter eines Drittanbieters fest.
  - Die Microsoft 365 E5, A5 und G5 enthalten Teams Telefonsystem, aber Organisationen müssen weiterhin eine [PSTN-Konnektivitätsoption auswählen](../pstn-connectivity.md).
