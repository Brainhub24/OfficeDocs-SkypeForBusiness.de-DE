Um **den Survivable Branch Appliance-FQDN** zu definieren, geben Sie den vollqualifizierten Dom채nennamen (Fully Qualified Domain Name, FQDN) (wie im Dns (Domain Name System) definiert) der Survivable Branch Appliance oder des Survivable Branch Servers in das **FQDN-Textfeld** ein.
