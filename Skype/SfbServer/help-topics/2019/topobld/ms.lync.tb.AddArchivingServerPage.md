# <a name="add-archiving-server"></a>Hinzufügen des Archivierungsservers
 
Zum Bereitstellen eines Archivierungsservers innerhalb des Topologieentwurfs müssen Sie den vollqualifizierten Domänennamen des Servers angeben, auf dem Sie die Archivierung bereitstellen möchten. Bevor Sie eine Topologie veröffentlichen, die die Archivierung und die Installation von Skype for Business Server umfasst, sollten Sie alle Voraussetzungen für die Bereitstellung der Archivierung erfüllt haben.
