# <a name="legacy-merge-add-expanded-edge"></a>Legacyzusammenführung – Hinzufügen einer erweiterten Edgekomponente
 
Skype for Business Server unterstützt nicht die Zusammenführung einer erweiterten Edgeserverkonfiguration von Communications Server 2007 R2 Office Legacy. Der erweiterte Edgeserver muss zunächst in einen konsolidierten Edgeserver umgewandelt werden, ehe die Zusammenführung der Edgeserverinformationen erfolgen kann.
