# <a name="legacy-merge-add-expanded-edge"></a>Legacyzusammenführung – Hinzufügen einer erweiterten Edgekomponente
 
Skype for Business Server unterstützt das Zusammenführen einer älteren Office erweiterten Edgeserverkonfiguration für Communications Server 2007 R2 nicht. Der erweiterte Edgeserver muss zunächst in einen konsolidierten Edgeserver umgewandelt werden, ehe die Zusammenführung der Edgeserverinformationen erfolgen kann.
