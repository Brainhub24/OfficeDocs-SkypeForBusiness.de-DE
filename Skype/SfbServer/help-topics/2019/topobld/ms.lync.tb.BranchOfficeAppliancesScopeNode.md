# <a name="branch-office-appliances-scope-node"></a>Filialanwendungen – Bereichsknoten
 
Der Knoten **Survivable Branch Appliances** enthält die Survivable Branch Appliances dieser Zweigstelle. Folgende Möglichkeiten stehen zur Verfügung:
  
- **Neue Survivable Branch Appliance**. Startet den Assistenten zum **Definieren neuer Survivable Branch Appliance,** mit dem Sie diesem Standort eine neue Survivable Branch Appliance hinzufügen.
    
Zusätzlich zur Survivable Branch Appliance-Option können Sie die folgenden allgemeinen Aufgaben ausführen:
  
- **Topologie**. Sie können diese Option verwenden, um eine neue Topologie zu definieren, eine vorhandene Topologie zu öffnen, die aktuelle Topologie herunterzuladen, eine Kopie der Topologie zu speichern, die Topologie zu veröffentlichen, eine Datenbank zu installieren, Komponenten aus früheren Versionen zusammenzuführen oder die Bereitstellung zu entfernen.
    
- **Ansicht**: Mithilfe dieser Option können Sie die Ansicht des Topologie-Generators anpassen, indem Sie die Elemente auswählen, die in den Ansichten der Konsolenstruktur ein- bzw. ausgeblendet werden sollen.
    
- **Hilfe**: Über diese Option greifen Sie auf die Hilfe für den aktiven Knoten zu.
