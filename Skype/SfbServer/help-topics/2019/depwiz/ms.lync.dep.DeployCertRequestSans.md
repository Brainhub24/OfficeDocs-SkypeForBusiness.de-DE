# <a name="certificate-request-additional-subject-alternate-names"></a>Zertifikatanforderung (zusätzliche alternative Antragstellernamen)
 
Die Option **Weitere alternative Antragstellernamen konfigurieren** bietet eine Möglichkeit, dem Zertifikat vor dessen Fertigstellung und Anforderung weitere alternative Antragstellernamen hinzuzufügen. Geben Sie zum Hinzufügen des alternativen Antragstellernamens dessen vollqualifizierten Domänennamen ein, und klicken Sie dann auf **Hinzufügen**. Wenn Sie einen alternativen Antragstellernamen hinzugefügt haben, den Sie löschen möchten, markieren Sie dessen Eintrag, und klicken Sie auf **Entfernen**. Klicken Sie zum Entfernen aller Einträge mit alternativen Antragstellernamen auf **Alle löschen**.
  
> [!NOTE]
> Die Schaltflächen **"Alle entfernen"** oder **"Alle löschen"** funktionieren nur für SAN-Einträge, die auf dieser Seite hinzugefügt werden. Die Schaltflächen haben keinen Einfluss auf Einträge mit alternativen Antragstellernamen, die sich derzeit auf dem Zertifikat befinden.
