# <a name="install-database-create-mirror-database-page"></a>Installieren einer Datenbank – Seite „Spiegeldatenbank erstellen“
 
Dies ist eine vorläufige Dokumentation und kann geändert werden. Leere Themen wurden als Platzhalter hinzugefügt.
  
Wenn Sie die SQL Server während des Veröffentlichungsprozesses nicht installieren möchten, installieren Sie sie mithilfe der **Datenbank** installieren mit einem Rechtsklick auf einen beliebigen Knoten im Topologie-Generator. Klicken Sie dazu mit der rechten Maustaste im fea-cs-topo-tool auf einen beliebigen Knoten, und wählen Sie den zu installierenden Spiegelspeicher aus. Klicken Sie auf **Einstellungen**, um erweiterte Einstellungen zu definieren.
