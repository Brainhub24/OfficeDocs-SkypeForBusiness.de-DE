# <a name="mobility-planning-tool"></a>Mobilitätsplanungstool
 
Das Mobilitätsfeature in Skype for Business Server 2015 unterstützt Skype for Business Funktionen auf mobilen Geräten.
  
Die Mobilitätsfunktionalität ist identisch mit der in Lync Server 2013. Weitere Informationen zur Vorbereitung Ihrer Umgebung auf Mobilität finden Sie unter [Plan for Mobility for Skype for Business Server](../../plan-your-deployment/mobility.md).
