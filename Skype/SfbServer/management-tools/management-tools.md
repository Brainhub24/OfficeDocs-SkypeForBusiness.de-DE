# <a name="skype-for-business-server-2015-management-tools"></a>Skype for Business Server 2015-Verwaltungstools
 
**Zusammenfassung:** Erfahren Sie mehr über die Dienstverwaltungstools in Skype for Business Server 2015.
  
Skype for Business Server 2015-Kommunikationssoftware (vormals Lync Server) bietet Chat-, Anwesenheits-, Konferenz- und Telefonielösungen, die Anforderungen an die Zusammenarbeit auf Unternehmensebene unterstützen. Die Tools zum Verwalten dieser Dienste sind flexibel und leistungsstark.
