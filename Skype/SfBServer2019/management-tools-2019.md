# <a name="skype-for-business-server-2019-management-tools"></a>Skype for Business Server 2019-Verwaltungstools
 
**Zusammenfassung:** Erfahren Sie mehr über die Dienstverwaltungstools in Skype for Business Server 2019.
  
Skype for Business Server 2019 bietet Sofortnachrichten-, Anwesenheits-, Konferenz- und Telefonielösungen, die Anforderungen an die Zusammenarbeit auf Unternehmensebene unterstützen. Die Tools zum Verwalten dieser Dienste sind sowohl flexibel als auch leistungsstark.

Sie ist mit KHI-Schwellenwerten aus den KHI-Ressourcen konfiguriert und kann an die individuellen Anforderungen Ihrer Bereitstellung angepasst werden.  <br/> |
