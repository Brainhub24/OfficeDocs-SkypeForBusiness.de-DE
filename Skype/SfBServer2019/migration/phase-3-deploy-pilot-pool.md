# <a name="phase-3-deploy-skype-for-business-server-2019-pilot-pool"></a>Phase 3: Bereitstellen Skype for Business Server 2019-Pilotpools

In diesem Abschnitt werden die erforderlichen Schritte zum Bereitstellen eines Pilotpools von Skype for Business Server 2019 behandelt. Für die Bereitstellung von Skype for Business Server 2019 müssen Sie den Topologie-Generator verwenden, um Ihre Topologie und die bereitzustellenden Komponenten zu definieren, Ihre Umgebung für die Bereitstellung der Skype for Business Server 2019-Komponenten vorzubereiten, den Topologieentwurf auf dem ersten Front-End-Server zu veröffentlichen und dann Skype for Business Server 2019-Software für die Komponenten für Ihre Bereitstellung zu installieren und zu konfigurieren. Nach Abschluss der Skype for Business Server 2019-Pilotpoolbereitstellung ist ein vorhandener Legacypool vorhanden.
